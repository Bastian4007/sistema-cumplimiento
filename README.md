# Sistema de Gestión de Cumplimiento Normativo

## 🎯 Objetivo V1
Construir un sistema que permita:

- Gestionar activos
- Gestionar requerimientos
- Gestionar obligaciones
- Controlar tareas
- Subir y descargar documentos
- Calcular vencimientos automáticamente
- Mostrar semáforo inteligente

## 🧱 Stack Tecnológico

- Laravel
- PostgreSQL
- Git / GitHub
